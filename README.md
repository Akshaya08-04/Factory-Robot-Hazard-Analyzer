# Factory Robot Hazard Analyzer

## Overview
The **Factory Robot Hazard Analyzer** is a Java-based application designed to identify and report potential safety hazards in automated factory environments. It helps engineers and safety managers monitor robot operations and prevent accidents.

## Features
- Detects hazardous robot movements and unsafe conditions
- Generates detailed safety reports
- Supports multiple robot configurations
- Easy integration with factory management systems

## Technologies Used
- Java (JDK 25)
- Git & Git Flow for version control
- Optional: GUI or CLI for user interaction
